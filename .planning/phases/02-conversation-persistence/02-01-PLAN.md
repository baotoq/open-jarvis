    Step 6 — Implement sqlitestore.go:
    - NewSQLiteConvStore(db *sql.DB) (*SQLiteConvStore, error) — runs migrate()
    - Schema (CREATE TABLE IF NOT EXISTS):
      ```sql
      CREATE TABLE IF NOT EXISTS conversations (
          id         TEXT PRIMARY KEY,
          title      TEXT NOT NULL DEFAULT '',
          created_at INTEGER NOT NULL,
          updated_at INTEGER NOT NULL
      );
      CREATE TABLE IF NOT EXISTS messages (
          id              INTEGER PRIMARY KEY AUTOINCREMENT,
          conversation_id TEXT NOT NULL REFERENCES conversations(id) ON DELETE CASCADE,
          role            TEXT NOT NULL,
          content         TEXT NOT NULL,
          position        INTEGER NOT NULL
      );
      CREATE INDEX IF NOT EXISTS idx_messages_conv_id ON messages(conversation_id, position);
      ```
    - Get(sessionID): SELECT messages WHERE conversation_id = sessionID ORDER BY position; return as []openai.ChatCompletionMessage
    - Set(sessionID, msgs): INSERT OR REPLACE into conversations (upsert); DELETE existing messages; INSERT all messages with position index
    - ListConversations(): SELECT conversations ORDER BY updated_at DESC
    - GetConversation(id): SELECT single conversation by id, return 404-style nil if not found
    - DeleteConversation(id): DELETE FROM conversations WHERE id = ? (CASCADE removes messages)
    - CreateConversation(id, title): INSERT conversation with created_at = updated_at = time.Now().Unix()
