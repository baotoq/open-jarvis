    1. Create src/internal/handler/chathandler.go. Handler responsibilities ONLY:
       - Set SSE headers: Content-Type: text/event-stream, Cache-Control: no-cache, Connection: keep-alive
       - Parse the request body into types.ChatRequest using httpx.Parse
       - Create ChatLogic and call l.StreamChat(&req, w)
       - On error, log with logx.WithContext — do NOT write headers again (stream may be partially written)

    2. Create src/internal/logic/chatlogic.go. Logic responsibilities:
       - NewChatLogic(ctx context.Context, svcCtx *svc.ServiceContext) — store ctx in l.ctx
       - StreamChat(req *types.ChatRequest, w http.ResponseWriter) error:
         a. context.WithTimeout(l.ctx, duration) for SAFE-03
         b. Get history from ConvStore; if empty, prepend system prompt message
         c. Append user message
         d. Assert w.(http.Flusher); return error if not supported
         e. Call AIClient.CreateChatCompletionStream with the assembled history
         f. Loop: recv token, skip if empty, write "data: TOKEN\n\n", flush
         g. On io.EOF: break loop, append assistant message to history, Set in ConvStore
         h. On other error (including context.DeadlineExceeded): write "data: [ERROR] MSG\n\n", flush, return error

       - Load config with conf.MustLoad from etc/config.yaml
       - Create rest.MustNewServer(c.RestConf)
       - Create svc.NewServiceContext(c)

       - server.Start()
