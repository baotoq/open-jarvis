  <behavior>
    StreamChat agentic loop:
    - Passes chatTools to every CreateChatCompletionStream call
    - During streaming, accumulates tool call fragments by Index (delta.ToolCalls, *int index)
    - After stream ends: if FinishReason == FinishReasonToolCalls, dispatch tools; else break
    - Tool dispatch: for each accumulated ToolCall, emit tool_call SSE event, check if shell requires approval, execute via Executor, emit tool_result SSE event, append role=tool message to history
    - Shell approval gate: if ShellTool.RequiresApproval(command), register channel, emit approval_request SSE event, select on ch and ctx.Done(); denied → ToolResult{Error: "user denied"}
    - The assistant message with ToolCalls populated MUST be appended to history before tool result messages
    - Loop terminates after MaxToolCalls iterations (reuse existing Config.MaxToolCalls)
    - All existing behavior preserved: session ID assignment, done event, conversation persistence

    SSE event formats (JSON, not bare text):
    ```
    data: {"type":"tool_call","id":"call_abc","name":"read_file","args":"{\"path\":\"foo.txt\"}"}
    data: {"type":"tool_result","id":"call_abc","content":"file contents"}
    data: {"type":"approval_request","approvalId":"uuid","tool":"shell_run","command":"ls -la"}
    ```

    Rewrite `src/backend/internal/logic/chatlogic.go`. Preserve the function signature and all existing session/persistence logic. Replace the single streaming call with a bounded agentic loop.

    Key implementation steps:

    1. Define `chatTools` as a package-level var (see interfaces above).

    2. Replace the single `CreateChatCompletionStream` call with a loop:
       ```go
       for iteration := 0; iteration < l.svcCtx.Config.MaxToolCalls; iteration++ {
           stream, err := l.svcCtx.AIClient.CreateChatCompletionStream(ctx,
               openai.ChatCompletionRequest{
                   Model:    l.svcCtx.Config.Model.Name,
                   Messages: history,
                   Tools:    chatTools,
               })
           ...
       }
       ```

    3. Inside the stream receive loop, accumulate tool call fragments:
       ```go
       toolCallAccum := map[int]*openai.ToolCall{}
       var finishReason openai.FinishReason
       for {
           resp, err := stream.Recv()
           if errors.Is(err, io.EOF) { break }
           ...
           delta := resp.Choices[0].Delta
           // Text tokens
           if delta.Content != "" { ... emit SSE token ... }
           // Tool call fragments
           for _, tc := range delta.ToolCalls {
               if tc.Index == nil { continue }
               idx := *tc.Index
               if _, ok := toolCallAccum[idx]; !ok {
                   toolCallAccum[idx] = &openai.ToolCall{ID: tc.ID, Type: tc.Type}
                   toolCallAccum[idx].Function.Name = tc.Function.Name
               }
               toolCallAccum[idx].Function.Arguments += tc.Function.Arguments
           }
           if len(resp.Choices) > 0 {
               finishReason = resp.Choices[0].FinishReason
           }
       }
       ```

    4. After stream ends, check finishReason:
       ```go
       if finishReason != openai.FinishReasonToolCalls {
           break // normal stop — exit loop
       }
       ```

    5. Convert map to ordered slice:
       ```go
       var toolCalls []openai.ToolCall
       for i := 0; i < len(toolCallAccum); i++ {
           if tc, ok := toolCallAccum[i]; ok {
               toolCalls = append(toolCalls, *tc)
           }
       }
       ```

    6. Append assistant message with ToolCalls to history (CRITICAL — must be before tool results):
       ```go
       history = append(history, openai.ChatCompletionMessage{
           Role:      openai.ChatMessageRoleAssistant,
           ToolCalls: toolCalls,
       })
       ```

    7. For each tool call, dispatch:
       - Emit tool_call SSE event
       - If name == "shell_run": check RequiresApproval; if true, run approval gate
       - Call svcCtx.Executor.Execute(ctx, tc.Function.Name, tc.Function.Arguments)
       - Emit tool_result SSE event
       - Append role=tool message to history

    8. Approval gate (extract to helper method `waitForApproval`):
       ```go
       func (l *ChatLogic) waitForApproval(ctx context.Context, w http.ResponseWriter, flusher http.Flusher, approvalID, command string) error {
           ch := make(chan bool, 1)
           l.svcCtx.ApprovalStore.Register(approvalID, ch)
           defer l.svcCtx.ApprovalStore.Delete(approvalID)
           // emit approval_request event
           event, _ := json.Marshal(map[string]any{"type":"approval_request","approvalId":approvalID,"tool":"shell_run","command":command})
           fmt.Fprintf(w, "data: %s\n\n", event)
           flusher.Flush()
           select {
           case approved := <-ch:
               if !approved { return errors.New("user denied command execution") }
               return nil
           case <-ctx.Done():
               return ctx.Err()
           }
       }
       ```

    Use `github.com/google/uuid` (already in go.mod) for approvalID generation.

    For tests, add mock types that satisfy AIStreamer to simulate tool_calls finish reason. The existing chatlogic_test.go already has mock infrastructure — extend it. Use NewServiceContextForTest from plan 02 to get a testable ServiceContext.

    PITFALL NOTES (from research):
    - Tool call args arrive fragmented — only unmarshal after finishReason == FinishReasonToolCalls
    - Tool result must be in history before re-calling model
    - Always select on both ch and ctx.Done() in approval gate

    ```go
    package handler

    import (
        "net/http"
        "open-jarvis/internal/svc"
        "open-jarvis/internal/types"
        "github.com/zeromicro/go-zero/rest/httpx"
    )

    func ApproveHandler(svcCtx *svc.ServiceContext) http.HandlerFunc {
        return func(w http.ResponseWriter, r *http.Request) {
            var req types.ApproveRequest
            if err := httpx.Parse(r, &req); err != nil {
                httpx.Error(w, err)
                return
            }
            if !svcCtx.ApprovalStore.Resolve(req.ApprovalID, req.Approved) {
                http.Error(w, "unknown approval ID", http.StatusNotFound)
                return
            }
            w.WriteHeader(http.StatusNoContent)
        }
    }
    ```
