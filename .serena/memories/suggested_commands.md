# Suggested Commands

## Go (Backend)
- `go build ./...` тАФ build all Go packages
- `go run ./cmd/...` тАФ run the Go service
- `go test ./...` тАФ run Go tests
- `go mod tidy` тАФ clean up go.mod/go.sum
- `go vet ./...` тАФ static analysis

## TypeScript/Next.js (Frontend)
- `npm install` тАФ install dependencies
- `npm run dev` тАФ start Next.js dev server
- `npm run build` тАФ production build
- `npm run start` тАФ start production server
- `npm run lint` тАФ run ESLint

## System (macOS/Darwin)
- `git`, `ls`, `find`, `grep` тАФ standard unix tools available
- `open` тАФ open files/URLs in macOS
